AI Newsfeed

Ett personligt AI-nyhetsflöde byggt för att inte missa det som faktiskt händer i branschen.

Vad är det här?

AI Newsfeed aggregerar nyheter från ledande AI-källor och sammanfattar dem automatiskt med Claude AI. Istället för att följa ett dussintal sajter och nyhetsbrev får du allt på ett ställe — skummbart, strukturerat och utan betalväggar.

Lägg till appen på hemskärmen för snabbaste upplevelsen — den fungerar som en installerad app, även offline.

Användarpulsen — vad praktiker bygger och diskuterar:

  • Simon Willison
  • Hacker News AI (endast högt röstade)

Hur det fungerar

  1. GitHub Actions hämtar nyheter fyra gånger om dagen (03:00, 09:00, 13:00, 18:00 svensk tid)
  2. Claude AI (Haiku) sammanfattar varje artikel automatiskt
  3. Artiklarna visas grupperade efter ämne

Tech stack

  • Frontend: HTML, CSS, JavaScript (statisk PWA, GitHub Pages)
  • Backend: Python, GitHub Actions
  • Databas: Google Firestore
  • AI: Anthropic Claude API (Haiku)
  • Hosting: GitHub Pages (gratis)
